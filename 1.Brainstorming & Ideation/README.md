# Brainstorming Phase

This folder contains all brainstorming activities for the Rising Water Level Prediction project.

## Files Included:
- Brainstorming Idea Prioritization
- Define Problem Statements
- Empathy Map

## Description:
In this phase, we analyzed the problem of rising water levels in rivers, lakes, and urban areas. Various ideas were generated to understand the root causes and possible solutions. The ideas were then prioritized based on feasibility and impact.

## Outcome:
- Clear understanding of the problem
- Initial solution directions identified
